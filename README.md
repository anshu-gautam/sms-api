### Steps to run
- Install Node `20.x.x`
- Clone the repo:
- `cd sms-api`
- `npm i`
- `touch .env`
- Add the values mentioned in the `.env.example` file
- `npm run dev`


### UI App:
Please visit [this](https://github.com/anshu-gautam/sms-ui) repo to setup the UI app
